# see-app-acessibilidade
# ♿ SEE - Mobilidade com Autonomia

O **SEE (Olhos que não veem, mas enxergam tudo)** é um projeto de aplicativo voltado para acessibilidade, com foco em ajudar pessoas com deficiência visual a utilizarem o transporte público com mais autonomia e segurança.

---

## 🎯 Objetivo

Facilitar a mobilidade urbana de pessoas com deficiência visual, fornecendo informações acessíveis sobre pontos de ônibus e navegação.

---

## 🚀 Funcionalidades

* 📍 Localização de pontos de ônibus próximos (GPS)
* 🔊 Alertas sonoros
* 📳 Feedback por vibração
* 🧭 Navegação acessível
* 🎙️ Suporte a comandos de voz (conceito)
* ♿ Interface adaptada para acessibilidade

---

## 🖥️ Interface

O projeto conta com uma interface desenvolvida com foco em acessibilidade:

* Alto contraste
* Texto ampliado
* Navegação por teclado
* Feedback visual e tátil
